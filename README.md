# 🎉 Smart Event Management System

An advanced futuristic **AI-powered Smart Event Management System** designed for intelligent event planning, booking management, guest monitoring, budget analytics, and premium celebration organization using modern web technologies.

---

# 📌 Project Overview

The **Smart Event Management System** is a modern digital event planning platform developed to simulate intelligent wedding, conference, party, and corporate event management infrastructure.

The platform provides:

* Smart event planning
* Guest management system
* Budget estimation dashboard
* Venue booking simulation
* Event analytics visualization
* Intelligent scheduling system

This project demonstrates modern frontend engineering, responsive dashboard architecture, event analytics visualization, and intelligent management system design using HTML, CSS, JavaScript, and Chart.js.
